# SEO Reports

Dated audit outputs for buildwithriz.com. Committed to git for historical tracking.

## Naming convention

```
seo-audit-YYYY-MM-DD.md          Full on-page / technical SEO audit
gsc-audit-YYYY-MM-DD.md          Google Search Console pull (queries, pages, CTR)
content-refresh-YYYY-MM-DD.md    Blog refresh plan (which posts to update and why)
```

Use ISO dates (YYYY-MM-DD) so files sort chronologically.
